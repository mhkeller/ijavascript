The IJavascript kernel is implemented by class Kernel in kernel.js and handlers_v4.js. This kernel makes use of module jp in jp.js to handle messages in the IPython/Jupyter protocol. The kernel also makes use of the Javascript sessions provided by module sm in sm.js. This module provides Javascript sessions by means of a node.js server implemented in sm_server.js.
